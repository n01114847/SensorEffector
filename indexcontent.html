  • 4. Soldering
    • NeoPixel Ring: Solder one end of a M-M connector to the NeoPixel Ring. Repeat this process for 5V, GND, and DI.
    • RFID Sensor: Place the bent or straight pin through the RFID sensor's pin-out holes, make sure to do this to refrain from technical problems. Solder one end of a M-M connector to the RFID sensor. Repeat this process for all pins except RQ.
  • 5. Mechanical Assembly
    • Raspberry PI 3: Plug power supply from an outlet to the Raspberry Pi 3. After booting up, complete the OS installation.
    • RFID Sensor: Connect one end of the connectors to the corresponding GPIO pin-out. Refer to diagram here and here
      • SDA connects to Pin 24. (GPIO08)
      • SCK connects to Pin 23. (GPIO11)
      • MOSI connects to Pin 19. (GPIO10)
      • MISO connects to Pin 21. (GPIO09)
      • GND connects to Pin 6. (Ground)
      • RST connects to Pin 22. (GPIO25)
      • 3.3v connects to Pin 1. (3.3V)
      • Referral Link: https://pimylifeup.com/raspberry-pi-rfid-rc522/
    • NeoPixel Ring: After completion, connect one end of the connectors to the corresponding GPIO pin-out. Refer to pin-out here:
      • DI connects to Pin 12.(GPIO18)
      • 5v connects to Pin 1. (It won't short)
      • GND connects to Pin 6. (Ground)
    • Breadboard: The breadboard is used to hold the two sensors that require the same Raspberry Pi GPIO pin. Refer to diagram here
  • 6. Power Up
    • Connect all these pins while the Raspberry is not yet booted up. Tip: power this up, make sure the GND pin is placed in first before the 3.3V pin is. Once all the configurations have been done, the Raspberry Pi can now be connected.
  • 7. Production Testing
    • After powering up the Raspberry Pi, the RFID sensor should be lit indicating that it is on and ready for scanning. Unfortunately, the NeoPixel ring has no production test other than unit testing.
  • 8. Unit Testing
    • To test the RFID, configurations must be made. Here is the step-by step guide:
      • Open up terminal on Raspberry Pi to setup Raspbian and Python, follow the commands used. (After reboot, reopen terminal again to proceed)
      • sudo raspi-config
      • sudo reboot
      • lsmod | grep spi (From here, check to see if spi_bcm2835. If not, refer to referral guide)
      • sudo nano /boot/config.txt (Add “dtparam=spi=on” to the bottom of the file and then save it by pressing CTRL + X and confirm)
      • sudo apt-get update
      • sudo apt-get upgrade
      • sudo apt-get install python2.7-dev
      • cd ~
      • git clone https://github.com/lthiery/SPI-Py.git
      • cd ~/SPI-Py
      • sudo python setup.py install
      • cd ~
      • https://github.com/mxgxw/MFRC522-python
      • cd ~/MFRC522-python
      • sudo python Read.py (Test your sensor by using the tag, if you get lines of code saying card detected, it works)
      • Referral Guide: https://pimylifeup.com/raspberry-pi-rfid-rc522/
    • To test the NeoPixel Ring, download the repository to your Raspberry PI: https://github.com/jgarff/rpi_ws281x.git
    • Refer to RFID to navigate on the terminal. When doing so, open the repository and run the python code:
    • cd ~/rpi_ws281x/python/examples/
    • sudo python strandtest.py
